# Python_Diwali_Sales_Analysis

🚀 Python Data Analysis Project – Diwali Sales Analysis

I completed a beginner-friendly data analytics project using Python to analyze Diwali sales data and uncover business insights that can improve customer experience and increase sales.

🔧 Tools & Libraries Used
Python
Pandas
NumPy
Matplotlib
Seaborn

📊 Project Objectives
Clean and preprocess sales data
Perform exploratory data analysis (EDA)

Identify top-performing customer segments
Analyze sales trends by:
Gender
Age group
State
Occupation
Product category
Generate insights to help businesses improve marketing and customer targeting

📈 Key Insights
Married women aged 26–35 years contributed the highest sales
Customers from states like Maharashtra, Karnataka, and Uttar Pradesh placed more orders
IT, Healthcare, and Aviation professionals were major buyers
Food, Clothing, and Electronics categories had high demand

This project helped me strengthen my skills in:

Data cleaning
Data visualization
Business insight generation
Real-world sales analysis using Python
